# Frontend — CarSales CRM

Aplicação React 19 + Vite + TypeScript + MUI do CRM de venda de carros.

Para executar o sistema completo (API + banco + frontend), use o [README da raiz](../README.md) com `docker compose up --build`.

## Desenvolvimento local

```bash
npm install
npm run dev
```

O Vite encaminha `/api` para `http://localhost:5000`. A API precisa estar rodando (veja [docs/MANUAL.md](../docs/MANUAL.md)).
